# Hillary Anampiu Portfolio

A static portfolio for Hillary Anampiu. It highlights LarrynetBilling, HomelinkGlobal, trading automation work, QuantumTradingFunds, and an experimental build lab.

## Local preview

Open `index.html` in a browser. No build step or dependencies are required.

## Deploy

This folder is ready for Vercel as a static site. Import the GitHub repository in Vercel and use the default settings: no framework preset, no build command, and output directory `.`.

Do not place billing-system secrets, databases, `.env` files, or MikroTik credentials in this repository.
